# ball-beam-filter-dashboard
Digital filter design and analysis dashboard for the Ball &amp; Beam system — FIR (moving average) and IIR (Butterworth) filters implemented in MATLAB with interactive visualization. Control II · Universidad del Magdalena · 2026-I
